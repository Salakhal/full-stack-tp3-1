# 🚀 Gestion d'Utilisateurs - Application Full-Stack

## 📋 Description

Application web full-stack moderne pour la gestion d'utilisateurs avec **Node.js**, **Express.js**, **PostgreSQL** et **EJS**. Cette application permet d'effectuer des opérations CRUD complètes (Create, Read, Update, Delete) sur une base d'utilisateurs avec une interface utilisateur élégante et responsive.

## ✨ Fonctionnalités

- ✅ **CRUD complet** - Créer, lire, modifier et supprimer des utilisateurs
- 🔍 **Recherche en temps réel** - Recherche dynamique des utilisateurs
- 📊 **Tableau de bord statistique** - Visualisation des métriques clés
- 🎨 **Design moderne** - Interface utilisateur avec Tailwind CSS
- 📱 **Responsive** - Adaptation à tous les appareils (mobile, tablette, desktop)
- ⚡ **Animations fluides** - Transitions et animations CSS élégantes
- 🛡️ **Sécurisé** - Protection contre les injections SQL et validation des données
- 🔔 **Notifications toast** - Feedback utilisateur en temps réel
- 🎭 **Modal d'édition** - Interface de modification intuitive

## 🛠️ Technologies Utilisées

### Backend
- **Node.js** - Environnement d'exécution JavaScript
- **Express.js** - Framework web pour Node.js
- **PostgreSQL** - Base de données relationnelle
- **pg** - Client PostgreSQL pour Node.js
- **express-validator** - Validation des données

### Frontend
- **EJS** - Moteur de templates
- **Tailwind CSS** - Framework CSS utilitaire
- **Font Awesome** - Icônes vectorielles
- **JavaScript vanilla** - Interactions côté client

### Outils de développement
- **Nodemon** - Rechargement automatique en développement
- **dotenv** - Gestion des variables d'environnement

## 🚀 Installation

### Prérequis

- **Node.js** (version 14 ou supérieure)
- **PostgreSQL** (version 12 ou supérieure)
- **npm** ou **yarn** (gestionnaire de paquets)

### Étapes d'installation

1. **Cloner le dépôt**
```bash
git clone https://github.com/votre-utilisateur/mon-projet-express.git
cd mon-projet-express

```
## 2. Installer les dépendances

```
npm install
```

## 3.Configurer PostgreSQL

Créez une base de données PostgreSQL :
```
CREATE DATABASE ma_base;

```
